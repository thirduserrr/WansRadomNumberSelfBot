# Wans Number Typer

Wans Number Typer is a small, modern, and sleek Python app that randomly types numbers between 100 and 50,000 where your cursor is. It logs all numbers in a JSON file to prevent duplicates and features a customizable GUI.

## Features

- Random number typing with JSON logging
- Adjustable typing speed
- Start/Stop buttons and hotkey (Ctrl+Shift+T)
- Tabbed GUI: Main and Settings
- Customize background, text, and button colors
- Clear JSON database button
- Sleek hover effects and rounded buttons
